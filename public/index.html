Version Control Workshop

Welcome to our Git & GitHub workshop! This simple page demonstrates your first step in the world of version control.

Getting Started

Clone this repository to start working:

$ git clone https://github.com/workshop/hello-github.git

Basic Git Commands

  1. Initialize a repository: git init
  2. Check status: git status
  3. Stage changes: git add filename or git add . for all files
  4. Commit changes: git commit -m "Your message here"
  5. Create a branch: git branch branch-name
  6. Switch branches: git checkout branch-name
  7. Push to remote: git push origin main
  8. Pull updates: git pull origin main

The Git Workflow

Remember the basic workflow: Edit → Stage → Commit → Push
